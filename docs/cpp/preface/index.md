# はじめに

プログラミング基礎講習会へようこそ！

プログラミング基礎講習会は、C++を用いた基礎的なプログラミングを、短期間で集中的に習得する講習会です。 大学からプログラミングを始める初心者が、各班での活動において最低限必要なプログラミング力を習得することを目指します。

全4回、一緒に頑張っていきましょう！
